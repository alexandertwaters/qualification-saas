# Obligation schema contract v1.0

This directory contains the authoritative Obligation Schema Contract v1.0.

## Authority and immutability

- This contract is frozen for v1.x.
- Fields and field meanings MUST NOT be removed, renamed, or redefined within v1.x.
- Additive changes that do not alter meaning MAY be introduced only via a new contract version directory.
- Any change that alters semantics requires a new major contract version (e.g., v2.0).

## Scope

This contract defines the canonical shape of a single obligation record emitted by the resolver.

## Non-goals

This contract does not define execution steps, verification methods, acceptance criteria, pass/fail outcomes, or compliance determinations.
