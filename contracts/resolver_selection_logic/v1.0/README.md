# Resolver Selection Logic Contract v1.0

This directory contains the authoritative Resolver Selection Logic Contract v1.0.

## Authority and immutability

- This contract is frozen for v1.x.
- Selection rules and guarantees MUST NOT be removed, renamed, or redefined within v1.x.
- Additive clarifications MAY be introduced only in a new contract version directory.
- This contract defines selection behavior only; it MUST NOT encode evaluation, reasoning, or compliance logic.

## Scope

This contract defines how obligation records are selected based on explicitly declared inputs.

## Non-goals

This contract does not define obligation meaning, execution steps, verification methods, acceptance criteria, or compliance determinations.
